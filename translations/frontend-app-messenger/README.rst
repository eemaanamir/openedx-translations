Frontend App Messenger
======================

A messaging micro-frontend application for Wikilearn, built from [frontend-template-application](https://github.com/openedx/frontend-template-application).

Overview
--------

This application provides a comprehensive messaging interface for Wikilearn, allowing users to send and receive messages, manage conversations, and interact through an inbox interface.

Features
--------

* **Inbox Management**: View and organize all conversations in one place
* **Conversation View**: Detailed conversation threads with message history
* **New Message Modal**: Compose and send new messages to users
* **Redux State Management**: Efficient state management with Redux Toolkit

Getting Started
---------------

Prerequisites
~~~~~~~~~~~~~

* Node.js (v16 or higher recommended)
* npm or yarn

Installation
~~~~~~~~~~~~

1. Clone the repository
2. Install dependencies:

   .. code-block:: bash

      npm install

3. Configure your environment (see ``module.config.js.example``)
4. Start the development server:

   .. code-block:: bash

      npm start

The application will be available at ``http://local.openedx.io:8080``
